A simple, responsive flexbox grid system.

  • Percentage Based Grid

  • Batch takes advantage of the benifits that flexbox has to offer. Vertical alignment, uniform cell height and no floats!

  • Simple Installation

  • Download the vanilla CSS or the SCSS source files. Read the installation instructions to add Batch to your project.

  • Extendable Units

  • Add more sizes to the $unit-sizes map to have the single-batch() mixin distill your custom unit classes. Requires Sass, Bourbon and Neat.
