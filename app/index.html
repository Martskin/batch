A simple, responsive, flexbox grid system.

  • Percentage Based Grid

  • Batch takes advantage of the flexbox layout mode. Providing vertical alignment, uniform cell height and no floats!

  • Easy To Use

  • Download the precompiled CSS or the SCSS source file. Read the installation instructions to add Batch to your project.

  • Extendable Cell Sizes

  • Add more sizes to the $cell-sizes map to have the small-batch() mixin distill your custom cell classes. Requires Sass and Bourbon.

Custom Cell Sizes

Create your own cell sizes by adding to the $cell-sizes Sass map in the SCSS source file. The key is the numerator and then the small-batch() mixin loops through the value array for each denominator. The key-value pair is used in the class name and is also used to calculate the width percentage. Additionally you can remove sizes from the list that you don't need in your project to lighten up the compiled CSS file.

  
    
      // Default sizes are 3rds, 4ths, 6ths, 8ths, 12ths and 24ths:
      // 1-1, 1-2, 1-3, 1-4, 1-6, 1-8, 1-12, 1-24
      // 2-3
      // 3-4, 3-8
      // 5-6, 5-8, 5-12, 5-24
      // 7-8, 7-12, 7-24
      // 11-12, 11-24
      // 13-24
      // 17-24
      // 19-24
      // 23-24
    

    $cell-sizes : (
      1   :  (1, 2, 3, 4, 6, 8, 12, 24),
      2   :  (3),
      3   :  (4, 8),
      5   :  (6, 8, 12, 24),
      7   :  (8, 12, 24),
      11  :  (12, 24),
      13  :  (24),
      17  :  (24),
      19  :  (24),
      23  :  (24)
    );

Installation Instructions

Precompiled CSS

Download the minified CSS file and reference the external style sheet in the head section of your HTML document. Or copy the styles from the dev CSS file into your project's stylesheet

  
    <head>
      <link href="batch.min.css" rel="stylesheet">
    </head>

SCSS Source File

Download the SCSS source file and import Batch after Bourbon in your main manifest file.

  
    @import "bourbon";
    @import "_batch.scss";
