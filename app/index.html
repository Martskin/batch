A simple, responsive, flexbox grid system.

  • Percentage Based Grid

  • Batch takes advantage of the flexbox layout mode. Providing vertical alignment, uniform cell height and no floats!

  • Easy To Use

  • Download the vanilla CSS or the SCSS source file. Read the installation instructions to add Batch to your project.

  • Extendable Units

  • Add more sizes to the $unit-sizes map to have the single-batch() mixin distill your custom unit classes. Requires Sass, Bourbon and Neat.

Custom Unit Sizes

Create your own unit sizes by adding to the $unit-sizes Sass map in the SCSS source file. The key is the numerator and then the single-batch() mixin loops through the value array for each denominator. The key-value pair is used in the class name and is also used to calculate the width percentage.

  
    
      // Default sizes - 3rds, 4ths, 6ths, 8ths, 12ths and 24ths:
      // 1-1, 1-2, 1-3, 1-4, 1-6, 1-8, 1-12, 1-24
      // 2-3
      // 3-4, 3-8
      // 5-6, 5-8, 5-12, 5-24
      // 7-8, 7-12, 7-24
      // 11-12, 11-24
      // 13-24
      // 17-24
      // 19-24
      // 23-24
    

    $unit-sizes : (
      1   :  (1, 2, 3, 4, 6, 8, 12, 24),
      2   :  (3),
      3   :  (4, 8),
      5   :  (6, 8, 12, 24),
      7   :  (8, 12, 24),
      11  :  (12, 24),
      13  :  (24),
      17  :  (24),
      19  :  (24),
      23  :  (24)
    );

Modifier Classes

Grid Gutter

Add the batch-g--gutter class to the batch-g grid wrapper element to add a gutter between the cells. Adjust the gutter size by changing the $batch-gutter Sass variable in the SCSS source file.

<!-- Example Markup Code -->

  
    <div class="batch-g batch-g--gutter">
     <div class="batch-u-1-3">
      Grid Cell Unit 1
     </div>
     <div class="batch-u-1-3">
      Grid Cell Unit 2
     </div>
     <div class="batch-u-1-3">
      Grid Cell Unit 3
     </div>
    </div>
