# Batch
A simple, responsive, flexbox grid system.

*Version 2.0.0*

### Basic Usage
```html
<div class="batch-g batch-g--border batch-g--gutter batch-g--padding">
	<div class="batch-c-1-1 batch-c-md-1-2 batch-c-lg-1-3">
		Grid Cell 1
	</div>
	<div class="batch-c-1-1 batch-c-md-1-2 batch-c-lg-1-3">
		Grid Cell 2
	</div>
	<div class="batch-c-1-1 batch-c-md-1-2 batch-c-lg-1-3">
		Grid Cell 3
	</div>
</div>
```
