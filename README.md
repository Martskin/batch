#Batch

A simple, responsive, flexbox grid system.

*This is currently a work in progress*

*Version 1.0.0*

###Basic Usage
``<div class="batch-u-1-1 batch-u-md-1-2 batch-u-lg-1-4"></div>``


###Build
``npm install``

``gulp``
